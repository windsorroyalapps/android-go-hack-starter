# Android Go Hack Starter

A blazing fast Android app starter using Go + Gomobile. Built by your Android/BMW hacking genius.

## Features
- Pure Go performance
- Gomobile bindings for Android
- Ready for native hacking & custom BMW integrations (OBD? why not)

## Quick Start
1. `go install golang.org/x/mobile/cmd/gomobile@latest`
2. `gomobile init`
3. Build with `gomobile build -target=android`
